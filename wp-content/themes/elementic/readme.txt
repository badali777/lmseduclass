Elementic is a lightweight, multi-purpose WordPress theme that combines powerful customization capabilities with excellent performance.

== Description ==

Elementic is a modern, lightweight, and highly customizable WordPress theme designed for blogs, businesses, portfolios, and news websites. It features advanced customizer options, multiple layouts, full Gutenberg support, WooCommerce compatibility, and an accessibility-ready design.

= Features =

* Responsive Design - Mobile-first approach ensures your site looks great on all devices
* Custom Colors - Choose your own color scheme with the customizer
* Multiple Layouts - Right sidebar, left sidebar, or no sidebar options
* Custom Header - Add your own header image
* Custom Logo - Upload your brand logo
* Gutenberg Ready - Full support for the block editor with custom block styles
* WooCommerce Support - Build an online store with full WooCommerce integration
* Translation Ready - Easily translate into any language
* RTL Support - Full right-to-left language support
* Accessibility Ready - WCAG 2.1 compliant for better accessibility
* SEO Optimized - Clean, semantic HTML5 markup
* Fast Loading - Optimized for Core Web Vitals
* Widget Areas - Multiple widget areas including sidebar and footer
* Custom Widgets - Recent posts with thumbnails and social links widgets
* Post Formats - Support for all standard post formats
* Sticky Header - Optional sticky header functionality
* Breadcrumb Support - Easy navigation for your users
* Footer Widgets - Up to 4 footer widget columns

= Customization Options =

* Layout Settings - Choose sidebar position and container width
* Blog Settings - Control excerpt length and post meta display
* Color Settings - Customize primary and secondary colors
* Typography Settings - Choose from hundreds of Google Fonts
* Footer Settings - Add custom copyright text

= Recommended Plugins =

* Contact Form 7 - For contact forms
* Yoast SEO - For better SEO optimization
* WooCommerce - For e-commerce functionality

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Elementic includes support for WooCommerce, Contact Form 7, and all major page builders including Elementor, Beaver Builder, and more.

= How do I customize the colors? =

Go to Appearance > Customize > Color Settings to change the primary and secondary colors of your theme.

= Can I change the layout? =

Yes! Go to Appearance > Customize > Layout Settings to choose between right sidebar, left sidebar, or no sidebar layouts.

= Is this theme translation ready? =

Yes, Elementic is translation ready and includes a .pot file for easy translation.

= Does this theme support RTL languages? =

Yes, Elementic has full RTL (right-to-left) language support.
